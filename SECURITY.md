# Security policy

Layered Graphics processes untrusted documents, archives, images, and fonts. Please do not disclose suspected vulnerabilities in a public issue.

## Reporting

Use [GitHub private vulnerability reporting](https://github.com/iamkaf/layeredgraphics/security/advisories/new). Include affected versions or commits, reproduction steps, impact, and any suggested mitigation. You should receive an acknowledgement within seven days.

## Supported versions

The project is experimental alpha software. Security fixes are applied to the latest `main` branch and the newest published prerelease when one exists; older snapshots are not maintained.

## Scope

Reports involving archive traversal, decompression or allocation limits, malformed manifests, asset integrity, image/font decoding, WASM/native boundary safety, denial of service, or unintended filesystem access are especially valuable.
