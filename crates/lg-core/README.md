# layered-graphics

The canonical Rust document, command, container, inspection, and authoritative rendering engine for [Layered Graphics](https://github.com/iamkaf/layeredgraphics).

This crate is experimental alpha software. See the repository [README](https://github.com/iamkaf/layeredgraphics#readme), [format specification](https://github.com/iamkaf/layeredgraphics/blob/main/docs/spec/kgfx-v1.md), and [runtime support policy](https://github.com/iamkaf/layeredgraphics/blob/main/docs/API_SUPPORT.md).
