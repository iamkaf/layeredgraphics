# Editable showcase

This example combines an original raster source with fill, image, text, blend, transform, asset, and ordering operations. The public `lg` CLI produces both the final PNG and the editable `.kgfx` document used on the project site.

```bash
./examples/showcase/build.sh
```

The source artwork is AI-generated; the layout, typography, compositing, document, and export are produced by Layered Graphics.
