# Code of Conduct

## Our standard

We are committed to a welcoming, constructive, and harassment-free project. Be respectful, assume good intent while addressing impact, give actionable technical feedback, and make room for people with different backgrounds and levels of experience.

Unacceptable behavior includes harassment, discrimination, sexualized attention, threats, sustained disruption, publishing private information, or personal attacks.

## Enforcement

Report conduct concerns privately to the project owner through the contact options on [@iamkaf's GitHub profile](https://github.com/iamkaf). Reports will be reviewed promptly and handled as confidentially as practical. Maintainers may edit or remove contributions, lock discussions, issue warnings, or temporarily or permanently restrict participation.

This code applies in project spaces and when someone publicly represents the project.
